optional

通知查询成功或失败的回调函数。已废弃。

弃用:
使用 searchRequest:didFailWithError: 代替。
参数
searchRequest发起的查询
errInfo错误信息
OK 用于表示未发生错误
INVALID_UER_KEY 用户key非法或过期
SERVICE_NOT_EXIST 请求服务不存在
INSUFFICIENT_PRIVILEGES 无权限访问此服务
OVER_QUOTA 请求超出配额
INVALID_PARAMS 请求参数非法
UNKOWN_ERROE:code 未知错误
